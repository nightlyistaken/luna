## Luna Bot

Luna is a discord bot made with harmony framework. It is a fun bot with many
features. It is still in development and is not ready for production use.

## Features

- Fun
- Utility
- And more!

## Running

```sh
deno run --allow-env --allow-read=. --allow-net main.ts
```
